# ProfeMatch Backend

API para la gestión de usuarios y moderación de ProfeMatch.

# Configuración inicial
1. Clonar el repositorio.
2. Instalar dependencias: "npm install"
3. Crear un archivo ".env" en la raíz con las siguientes variables:
   - PORT=3000
   - DB_HOST=localhost
   - DB_USER=root
   - DB_PASSWORD=
   - DB_NAME=profematch_db
4. Importar el archivo "database/profematch_db.sql" en tu MySQL local.
5. Iniciar el servidor: "node server.js"
